186 斷食 Start !

Posted on 2024-06-28 In 運動生活

Fasting


這個計畫從 2024/06/24 開始，會到什麼時候結束我就不清楚了
所以先在開頭做一個紀錄。

起因嘛… 長話短說好了今年初開始運動量變少 食量照舊
一回過神來 已經胖了約4kg了 60 => 64

想說原本就有在執行168，跟我的日常生活幾乎重疊執行起來無壓力
平日中午12點吃午餐，早餐沒吃 回推起來，16小時是昨天的晚上8點
但竟然還是變胖 ! 仔細記錄8小時內到底都吃了什麼 沒想到晚上
一小時左右的運動時間結束後，除了練完的正餐 依然持續的亂吃小零食
自以為有運動?

某天發現App store上有不少斷食的APP，下載了一個廣告少一點的，
想說心一橫那乾脆來執行186吧，看能不能壤一直亂吃小零食的習慣給改掉

Daily plan

  • 當日晚間10點 用餐結束時間，直至18小時後
  • 翌日下午4點 用餐時間開始，直到晚間10點

下午4點到晚上10點真的很符合我的日常生活，平日練車練跑 約略在晚間 7:30~8:30開始
下午4點吃飯 消化完畢也大約6點多，不用像先前 趕著回家 趕著吃飯 趕著消化 趕著休息 趕著運動

整體來說非常好? 如果管得住自己?

Daily log

  • 2024/06/24 第一天 64.3kg
    • 第一次嘗試在有心理預期的狀態下超過中午12點沒吃飯，大約在下午2點多感覺餓了點但精神上沒有不好反而感覺精神更好了，但4點一到吃飯時狼吞虎嚥的 可能也因為這樣在大約5點多整個人昏昏欲睡有點糟糕。
  • 2024/06/25 第二天 63.5kg
    • 一直到下午4點 飢餓狀態、精神狀態 都處於正常狀態 不疲憊 不是特別飢餓，這天的午餐也正常的緩緩的吃 直到5:30精神尚可 稍微疲勞 應該是平常下班狀態的疲勞吧。

後面就待續嘍，這個計畫應該可以來個周更新。
